# 🌧️ 惊天魔盗团停雨游戏

这是一个基于 React 和 Vite 构建的互动天气控制应用，灵感来源于电影《惊天魔盗团》中的停雨场景。

prompt credit: 小红书 --- 苏格拉没有底

## 🌐 在线访问

项目已部署到 GitHub Pages，可通过以下链接访问：
https://pointbreaker.github.io/horseman_show/

## 🚀 本地运行

**环境要求：** Node.js

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 在浏览器中打开 http://localhost:3000 查看应用

## 📦 构建部署

项目使用 GitHub Actions 自动构建并部署到 GitHub Pages。当推送到 main 分支时，会自动触发部署流程。
